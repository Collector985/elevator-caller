CURRENT FLOOR INDICATOR
-----------------------
When elevator is at floor 15:
  ┌══┐  ← Purple/highlighted background
  │15│
  └══┘

When NOT at floor:
  ┌──┐  ← Dark gray background
  │14│
  └──┘

BUTTON STATES
-------------
Normal:   [UP]   Green background
Active:   [UP]   Yellow background (call placed)
Normal:   [DN]   Red background
Active:   [DN]   Yellow background
Normal:   [LD]   Orange background
Active:   [LD]   Yellow background

LAYOUT DIMENSIONS
-----------------
Screen:           480 x 800 pixels
Main panel:       460 x 720 pixels
Nav bar:          480 x 60 pixels
Floor box:        40 x 40 pixels
Button:           50 x 40 pixels
Row height:       48 pixels
Column spacing:   15 pixels between columns

ADVANTAGES OF THIS DESIGN
--------------------------
✓ All 30 floors always visible (no scrolling needed)
✓ Current floor clearly highlighted
✓ No wasted space on animated car graphic
✓ Larger, easier-to-tap buttons
✓ Cleaner, more professional look
✓ Instant visual feedback on elevator position
✓ More screen real estate for buttons
✓ Better for touch interface (no tiny targets)
